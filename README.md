# icc2021_college
# 問題描述
完成一地理圍籬(geofence)系統，本系統使用 6 顆接收器在平面上建構出虛擬圍籬，地
理圍籬系統依據接收器的座標及待測物體的座標來分析待測物體是在圍籬內或圍籬外。
